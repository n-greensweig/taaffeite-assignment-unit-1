# 4 - What is a quick way to create a new folder?

# 5 - What output would you expect when you run the command 'ls' in an empty folder? What about a folder containing files?
One would expect a blank output when running 'ls' in an empty folder and a list of file names when 
running 'ls' in a folder containing files.

# 8 - Why would you run the command 'git status'?
One might run the 'git status' command to ensure a recent attempt to commit one's code changes to git

# 9 - What does the command 'git add .' do?
The 'git add .' command saves all code changes since the previous commit to a new commit.
